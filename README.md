# CSS Position Demo

A simple HTML project demonstrating different CSS positioning properties.

## Features

- Static Position
- Relative Position
- Absolute Position
- Sticky Position
- Scroll demonstration

## Technologies Used

- HTML5
- CSS3

## Project Structure

```
css-position-demo/
│
├── positionEx.html
└── README.md
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/css-position-demo.git
```

2. Open the project folder.

3. Double-click `positionEx.html`

or

Open it using VS Code and install the **Live Server** extension.

Then click **Go Live**.

## Learning Objectives

This project helps understand:

- CSS Position Property
- position: static
- position: relative
- position: absolute
- position: sticky
- Top, Bottom, Left, Right properties
- Scrolling behavior
